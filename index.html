After obtaining my Bachelors of Science in Computer Science degree from TLU in May 2017, I spent some gaining experience in Full-Stack development utilizing both front-end and back-end technologies including HTML, CSS, JavaScript, Java, Python, and MySQL.

I am passionate about what I do and I never settle with what I know.

Projects


Collision-Avoiding Remote-Controlled Car

- Project Scope: Created a robotic device that utilized a single-board computer embedded into a motor vehicle to allow a user control of the vehicle through the use of a multiple different input devices. These include a website, a bluetooth controller, a radio controller, and over a wireless network. The vehicle drove autonomously though the use of object recognition that utilized a camera and multiple different sensors to gather surrounding environmental data.

  • Python, Arduino IDE, HTML5, JavaScript, CSS and Shell

  • https://github.com/chjlarson/Collision_Avoiding_RC_Car

  • Successful communication between hardware components and software

  • Utilizes object-recognition software to aid the collision-avoidance algorithm.

  • Utilizes machine learning to aid the collision-avoidance algorithm.

  • Can be run autonomously (Navigates on its own and explores its surroundings)

  • Can be controlled using a variety of controllers including the original controller, Bluetooth controller, web client, command-line, or Android application.

  • Continuing further development. Github repository reflects a single tested and complete working solution from the first prototype. Currently, there are 2 other variations being developed and tested.
